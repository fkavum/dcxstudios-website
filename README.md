# DcxStudios Website

A modern, responsive static website for DcxStudios game development studio.

## Structure

- `index.html` - Main homepage with hero, about, features, projects, contact, and footer sections
- `games/` - Folder containing individual game detail pages
  - `game-1.html` - Template game detail page
- `styles/main.css` - All styling and responsive design
- `assets/` - Directory for images and icons
  - `images/` - Game screenshots and background images
  - `icons/` - Game icons and favicon

## Features

- Modern dark theme with professional game studio aesthetic
- Fully responsive design (mobile, tablet, desktop)
- Clean typography using Inter font family
- Smooth transitions and hover effects
- Game card grid with hover overlays
- Individual game detail pages with gallery, description, and features

## Customization

### Replacing Placeholders

1. **Game Cards**: Edit the project cards in `index.html` (Other Projects section)
   - Replace placeholder images with your game screenshots
   - Update game titles and descriptions
   - Update links to point to your game detail pages in `games/` folder

2. **Game Detail Pages**: 
   - Copy `games/game-1.html` for each new game
   - Replace game icon, title, and subtitle
   - Add your screenshots to the gallery
   - Update description and features
   - Add video embed if available

3. **Contact Information**: 
   - Update email address in Contact section
   - Add your social media links

4. **Footer Links**: 
   - Add Privacy Policy and Terms of Service pages
   - Update footer links accordingly

5. **Images**: 
   - Replace placeholder images with your actual game assets
   - Recommended sizes:
     - Game cards: 600x400px
     - Screenshots: 800x600px
     - Game icons: 120x120px

## Color Scheme

The website uses a dark theme with the following color palette:
- Primary Background: #0F172A
- Secondary Background: #1E293B
- Accent Colors: #6366F1 (primary), #8B5CF6 (secondary)
- Text: #F8FAFC (primary), #CBD5E1 (secondary)

Colors can be customized in `styles/main.css` using CSS variables.

## Browser Support

- Modern browsers (Chrome, Firefox, Safari, Edge)
- Mobile browsers (iOS Safari, Chrome Mobile)
